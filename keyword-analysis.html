10Opportunity Scoring

엔트로피 가중치 기법(Entropy Weight Method)으로 지표별 가중치를 데이터 분산에 기반해 자동 산출하고, 가중합으로 0~100점 기회점수를 계산했습니다.

자동 산출된 지표 가중치
Entropy Weight (분산이 클수록 정보량이 많다고 판단)

⚠️ 가중치 해석 주의: 엔트로피 가중치는 각 지표의 '통계적 분산'만을 기준으로 자동 산출되어, 총검색수·쇼핑성지수처럼 값의 편차가 큰 지표에 가중치가 쏠리고(각 52.2%·25.6%), 경쟁강도·CPC·클릭경쟁률처럼 편차가 상대적으로 작은 지표는 실제 비즈니스 중요도와 무관하게 가중치가 0.2~0.4% 수준으로 거의 반영되지 않습니다. 즉 현재 기회점수는 사실상 '검색량+쇼핑성지수' 위주의 점수이며, 경쟁강도가 낮다는 이유만으로 상위에 오르지는 않습니다. 이 가중치를 조정하려면 Top 20·추천등급·원본 테이블의 점수를 모두 재계산해야 일관성이 유지됩니다.
